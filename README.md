# rm-character-ui

Small Angular (v21) UI that consumes the Rick and Morty API endpoint:

- `GET https://rickandmortyapi.com/api/character/{id}`

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:4200/`.


## Note
This project uses TypeScript ~5.9.x because Angular 21.1.x requires TypeScript >=5.9 <6.0.
